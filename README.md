# Null_Sect0r

Репозиторий содержит сборник детальных райтапов по различным CTF соревнованиям от команды **Null_Sect0r**. Каждый writeup включает пошаговый разбор задач, подходы к решению, а также ссылки на используемые инструменты или дополнительные материалы.​

### Как использовать
Откройте нужный райтап в папке writeups — все решения сопровождаются поясняющими комментариями и иногда скриптами.
Некоторые writeup'ы содержат материалы для практики.

### Для кого этот репозиторий
Для новичков, которые хотят разобраться с типовыми задачами CTF.
Для участников соревнований в поисках идей по разбору сложных задач.
